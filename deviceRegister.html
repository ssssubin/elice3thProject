[ Device 등록 과정 ]

1. Device(ESP32): AP 모드 활성화 -> Device 자체 WiFi 생성

-> 이 때, 사용자는 'ESP32_' 뒤에 적힌 기기 ID를 기억한 후, 기기 등록 페이지의 MAC ADDRESS 부분에 기입해야 함

2. 사용자가 Device WiFi 연결 시도 -> Device가 생성한 웹 페이지로 접속

-> WIFI SSID에는 사용자 와이파이의 이름을, WIFI PASSWORD에는 사용자 와이파이 비밀번호를 입력하여 연결

3. Device는 사용자가 입력한 정보를 받고 사용자의 WIFI와 연결됨

4. Device 등록 시, 사용자는 Device의 초기 데이터(기기ID, 기기명, 식물 이름, 온도/습도/지습 범위) 입력하여 등록
